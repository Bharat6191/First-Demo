Status types: 

untracked:
new files that git doesn't yet tracked

modified:
changed

staged:
File is ready to be commited

unmodified:
unchanged

Trank->
changed (MODIFIED) | Added new File (UNTRACKED) -> Add (STAGED) -> Commit(UNCHANGED)

Add-> add changes in git staging area
    git add . -> to add all the files from the current working directore to the staging area
    git add "File name" -> to add the perticular file in the git staging area
Reset: To unstaged the file
Commit-> commit the changes or the record of changes
